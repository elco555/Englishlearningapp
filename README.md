# LearningApp - אפליקציה ללימוד שפות

אפליקציית אינטרנט אינטראקטיבית ללימוד שפות (אנגלית, צרפתית, ספרדית וגרמנית) המבוססת על כרטיסיות זיכרון (Flashcards) ומבחנים מהירים.

## ✨ תכונות עיקריות
- **מגוון שפות:** תמיכה בלימוד אוצר מילים בסיסי באנגלית, צרפתית, ספרדית וגרמנית.
- **שיטת לימוד דו-שלבית:** 1. למידה באמצעות כרטיסיות הכוללות תרגום ודוגמאות משפטים.
  2. מבחן (Quiz) רב-ברירה בסיום כל שיעור לבדיקת הידע.
- **מעקב התקדמות:** מערכת Streak (רצף למידה) ושמירת שיעורים שהושלמו ב-LocalStorage של הדפדפן.
- **ממשק משתמש מודרני:** עיצוב נקי וצבעוני המותאם לעבודה בעברית (RTL).

## 🛠️ טכנולוגיות
- **React:** ספריית ה-Frontend לפיתוח הממשק.
- **Lucide-React:** לשימוש באייקונים וקטוריים.
- **Tailwind CSS:** עבור העיצוב והרספונסיביות.
- **JavaScript/TypeScript:** לניהול הלוגיקה של הנתונים והאפליקציה.
